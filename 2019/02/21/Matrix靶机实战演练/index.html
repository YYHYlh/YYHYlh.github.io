实战演练

首先配置环境，在vulnhub下载下来后，在电脑安装了VirtualBox的情况下直接双击ovf文件，即可自动在Virtualbox里进行安装，但是需要提前配置好虚拟网卡和DHCP服务器。

配置完成后打开虚拟机即可。打开后可以看到如下界面，即配置完成。

配置好就可以开始实战了。
首先确定了目标地址是服务器后的第一台主机，即IP为192.168.56.101。用nmap扫一下。

服务器开了一个22的ssh端口和两个web端口，分别curl80端口和31337端口，在31337端口的回显里发现如下信息

尝试用base64解码，得到如下明文

echo “Then you’ll see, that it is not the spoon that bends, it is only yourself. “ > Cypher.matrix

看起来像是一句命令，把一句话写入了一个文件Cypher.matrix。在服务器上寻找此文件，输入http://192.168.56.101:31337/Cypher.matrix，成功下载到这个文件，打开后内容如下

猜测是一段brainfuck代码，使用在线网站https://copy.sh/brainfuck/进行解码，解码得到如下文字

猜测guest为登陆ssh的用户名，k1ll0rXX为密码，其中最后两位需要爆破。
首先用python生成字典

输出到dic.txt中，然后使用hydra进行爆破

爆破得到密码为k1ll0r7n。
ssh登陆成功，然后尝试了一些命令，发现很多被限制或是过滤了。
